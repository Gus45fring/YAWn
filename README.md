# YAWn
<p>yet another wallpaper manager, made by me with rust and egui as a starter project to learn rust. It uses hyperctl.
</p>

## Features
<ul>
  <li>YAWn can change your desktop background.</li>
  <li>YAWn supports animated wallpapers</li>
  <li>YAWn can automatically change your wallpaper based on the time of day, using .json files to go along with the images or videos.</li>
  <li>Lots more features regarding .json files. Look at the docs.</li>
</ul>
